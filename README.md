# Find-MfuClients.ps1

Скрипт сканирует указанные подсети и ищет компьютеры, на которых настроен принтер (МФУ) с заданным IP. Результаты записываются в CSV.

## Требования

Достаточно **Windows PowerShell 5.1** (идущего в составе Windows). Параллельное выполнение реализовано через `Start-Job`.

## Настройка

В начале файла задаются:

- `$printerIP` — IP-адрес искомого принтера/МФУ
- `$output` — путь к результирующему CSV-файлу
- `$subnets` — массив подсетей для сканирования (например, `192.168.100.`, `192.168.101.`)
- `$maxParallel` — максимальное количество одновременных проверок

## Запуск

```powershell
.\Find-MfuClients.ps1
```

или:

```powershell
powershell -ExecutionPolicy Bypass -File .\Find-MfuClients.ps1
```

## Результат

- В консоли отображается прогресс и сообщения о найденных принтерах или ошибках доступа.
- В CSV записываются строки: `Computer,Printer,Port` — компьютер, имя принтера и порт.
